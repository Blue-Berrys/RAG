# CookRAG-Go - 企业级RAG系统

**项目概述**: 纯Go实现的企业级检索增强生成(RAG)系统，采用混合检索架构，通过智能路由自动选择最优策略，支持342份菜谱的知识图谱问答。

**核心亮点**:

1. **算法层 - RRF融合 + 自适应权重**
   实现RRF算法融合向量与BM25检索结果，根据查询复杂度自适应调整权重(简单0.3:0.7、复杂0.8:0.2)，使用goroutine并行提升性能

2. **架构层 - 智能路由 + 知识图谱**
   基于多维分析自动选择GraphRAG/Hybrid/Vector策略，构建Neo4j知识图谱(878实体+3449关系)支持多跳关系推理

3. **工程层 - 优雅降级 + 可观测性**
   集成Milvus/Neo4j/Redis实现连接失败时优雅降级，实现自定义span追踪系统覆盖7个核心模块，支持Prometheus监控和日志文件输出

- **AI服务**: 智谱AI、百度千帆、阿里DashScope
- **NLP**: jieba-go
- **监控**: Prometheus、自定义链路追踪
